# markdown-2-gost-word

`markdown-2-gost-word` — инструмент для конвертации Markdown-документов в
Microsoft Word `.docx`, оформленный по правилам РФ ГОСТ.

Проект нужен для автоматизации рутинной стилизации учебных и отчетных документов:
курсовых, дипломных работ, рефератов, отчетов по практике и других материалов, где
важны структура, отступы, стили, заголовки, списки, таблицы и единообразное
оформление.

## English Summary

`markdown-2-gost-word` converts Markdown documents into Microsoft Word `.docx`
files formatted according to Russian GOST-style academic/report requirements. The
goal is to reduce manual formatting work and make document styling configurable for
different rule sets.

## Идея проекта

Markdown удобен для написания текста: он легкий, читаемый и хорошо подходит для
версирования. Word нужен на финальном этапе, когда документ должен соответствовать
требованиям оформления.

Этот проект связывает оба подхода:

1. Автор пишет документ в Markdown.
2. Приложение разбирает Markdown на логические секции.
3. Секции преобразуются в структуру Word-документа.
4. К результату применяются правила оформления под ГОСТ.
5. На выходе создается `.docx`, готовый к дальнейшей проверке и сдаче.

## Гибкие правила оформления

Проект планируется развивать так, чтобы правила форматирования можно было менять
динамически под разные требования:

- разные варианты РФ ГОСТ;
- методички вузов и кафедр;
- правила для курсовых, дипломов и отчетов;
- локальные требования к отступам, шрифтам, заголовкам и структуре документа.

Изначальный фокус — документы учебного и отчетного характера: курсовые, дипломные
работы и отчеты, оформляемые по требованиям РФ ГОСТ.

## Поддерживаемые элементы Markdown

Целевой набор возможностей включает базовые элементы, которые обычно нужны при
подготовке курсовых, дипломов и отчетов:

- заголовки и иерархия разделов;
- обычные абзацы;
- нумерованные и маркированные списки;
- таблицы;
- изображения и подписи к ним;
- блоки кода и листинги;
- цитаты и примечания;
- ссылки;
- содержание и структурные разделы документа;
- список источников.

- C++ / C++/CLI;

- MSVC v143;
- Microsoft Word Open XML формат;
- `DocumentFormat.OpenXml` через bundled DLL в `thirdparty/openxml/net46`.

## Сборка

### Через Visual Studio

1. Откройте `markdown-2-gost-word.sln` в Visual Studio 2022.
2. Выберите конфигурацию `Release` или `Debug`.
3. Выберите платформу `x64` или `x86`.
4. Соберите проект через `Build Solution`.

### Через Developer PowerShell

Запускайте команды из корня репозитория в Visual Studio Developer PowerShell:

```powershell
nuget restore .
msbuild /m /p:Configuration=Release .
```

При необходимости можно указать платформу явно:

```powershell
msbuild /m /p:Configuration=Release /p:Platform=x64 .
```

## Запуск

На текущем этапе приложение запускается как консольная программа с интерактивным
меню:

```text
markdown-2-gost-word
1. Create test DOCX with OpenXML
2. Read md file
0. Exit
```

Текущий режим используется для проверки OpenXML-интеграции и первичной обработки
Markdown-файла.

В дальнейшем планируются несколько режимов работы:

- CLI-режим для автоматической конвертации из командной строки;
- GUI-режим для выбора файлов, профиля оформления и генерации Word-документа;
- режим выбора или настройки набора правил форматирования.

## Планируемый CLI-формат

Один из целевых вариантов запуска:

```powershell
markdown-2-gost-word input.md output.docx --rules gost-rf
```

Финальный интерфейс командной строки может измениться по мере развития проекта.

## Структура проекта

- `markdown-2-gost-word/markdown-2-gost-word.cpp` — точка входа и консольное меню.
- `MdSection` — логическая секция Markdown-документа.
- `MdSectionParser` — разбор Markdown-текста на секции.
- `MdSectionRule` — правила определения и преобразования секций.
- `MdSectionConverter` — конвертация Markdown-секций в Word/GOST-представление.
- `GostWordSection` — модель секции Word-документа с ГОСТ-оформлением.
- `OpenXmlService` — слой генерации `.docx` через OpenXML.
